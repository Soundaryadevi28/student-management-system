# 🎓 Student Management System

## 📌 Overview
This project is a simple Student Management System built using Python. It helps to store and manage student details like name and age using basic data structures.

## 🚀 Features
- Add new student details
- View all students
- Simple menu-driven program
- Beginner-friendly project

## 🛠️ Technologies Used
- Python

## 📂 Project File
- student management system.py

## ▶️ How to Run
1. Install Python on your system
2. Download this project
3. Open terminal or command prompt
4. Run the file using:
   python student management system.py

## 📸 Sample Output
- Enter name
- Enter age
- Student added successfully
- Display student list
